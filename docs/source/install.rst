============
Installation
============

Normal User
###########

.. code-block:: bash
    pip install torchfitter


Developer
#########

To install as developer you must use -e and replicate the environment in which
the library was built.

.. code-block:: bash
    git clone https://github.com/Xylambda/torchfitter.git
    pip install -e torchfitter/. -r torchfitter/requirements-dev.txt
